# 图书馆管理系统
基于 ssm 框架开发，主要提供 web 端和 mobile 端的 api 接口，适合初学者。

## versions
Java Compiler Version：1.7
Tomcat Version : 9.0
MySQL Version : 8.0.12
Ecplise Version : 2020-12 (4.18.0)

## 主要模块
### web
1. 图书分类 CRUD & import
2. 图书管理 CRUD & import & image upload
3. 管理员管理&读者管理

### mobile
1. 用户基本操作如 登录注册
2. 用户借阅书籍、添加借阅车、生成相应记录等
3. 书籍分类、书籍展示与排序
